# Zach Zhao Academic Personal Website

This is a small static academic homepage inspired by a minimalist faculty-style personal site.

## Files

- `index.html`: homepage with profile, bio, contact links, and CV link
- `research.html`: research interests, selected publications, and projects
- `personal.html`: personal notes and interests
- `styles.css`: shared responsive styling
- `Zach_Zhao.pdf`: CV file linked from the homepage

## Edit

The site has been filled with information extracted from `Zach_Zhao.pdf`.
Potential future edits:

- Add a real portrait image in place of the current placeholder motif.
- Add Google Scholar, GitHub, ORCID, or project links once available.
- Replace submitted/accepted publication placeholders with final citation details.

## Preview

Open `index.html` directly in a browser, or run a local static server from this folder.
